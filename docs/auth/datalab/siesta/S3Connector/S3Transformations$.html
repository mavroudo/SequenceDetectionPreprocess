auth.datalab.siesta.S3Connector

S3Transformations 

object S3Transformations

This class is responsible to transform the loaded data from S3 into RDDs to processed by the main pipeline. Then transform the RDDs back to Dataframes, in order to utilize the spark api to store them back in S3. Therefore there are 2 transformations for each table, one to load data and one to write them back.

  16.  def transformCountToDF(counts: RDD[Count]): DataFrame

    The required format is [event_typeA, List of statistics], where each statistic has the structure: [event_typeB, total_duration, number_of_completions, min_duration, max_duration].

    The required format is [event_typeA, List of statistics], where each statistic has the structure: [event_typeB, total_duration, number_of_completions, min_duration, max_duration]. That is for each event type pair, we maintain the following statistics:

    • Total duration: sum of all the timestamp differences for each occurrence of this event type
    • Total completions: the total number of appearances of this event in the log database
    • Min duration: the minimum difference between the two timestamps of the events for this event type pair
    • Max duration: the maximum difference between the two timestamps of the events for this event type pair
