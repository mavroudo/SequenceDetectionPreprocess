auth.datalab.siesta.S3Connector

S3Connector 

class S3Connector extends DBConnector

This class handles the communication between SIESTA and S3. It inherits all the signatures of the methods from auth.datalab.siesta.BusinessLogic.DBConnector and overrides the reads and writes to match S3 properties. S3 stores data in parquet files. These files allow for objects to be stored and some indexing that enables efficient query. The drawback is that parquet files are immutable and thus in order to append new records to a preexisting parquet file we have to rewrite the whole file.
